# Electromagnetism Project

A comprehensive educational and computational project exploring the principles and applications of electromagnetism.

## Project Overview

This project covers fundamental concepts of electromagnetism including:
- Electric fields and forces
- Magnetic fields and forces
- Electromagnetic induction
- Wave propagation
- Applications in modern technology

## Key Concepts

### Electric Fields
- Electric field generated by point charges
- Superposition principle
- Gauss's Law
- Electric potential and potential energy

### Magnetic Fields
- Magnetic field from current-carrying wires
- Ampère's Law
- Lorentz Force
- Magnetic flux and induction

### Electromagnetic Waves
- Maxwell's equations
- Wave equation solutions
- Electromagnetic spectrum
- Radiation and propagation

## Getting Started

### Prerequisites
```bash
pip install -r requirements.txt
```

### Running Examples
```bash
python examples/point_charge_field.py
python examples/solenoid_simulation.py
python examples/em_wave_propagation.py
```

### Interactive Notebooks
Open Jupyter notebooks in the `notebooks/` directory to explore concepts interactively.

## Core Modules

### electric_field.py
Implements electric field calculations:
- Point charge field calculations
- Field superposition
- Potential calculations

### magnetic_field.py
Handles magnetic field computations:
- Current element contributions
- Ampère's Law applications
- Lorentz force calculations

### electromagnetic_wave.py
Simulates electromagnetic wave behavior:
- Wave equations
- Propagation in different media
- Interference and diffraction
